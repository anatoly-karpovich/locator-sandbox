Playwright · Beginner trainings

These trainings are designed to build correct mental models for Playwright locators. You are not learning “how to find elements” — you are learning how to make decisions you won’t regret in a real test suite.

Strict API usage
Tasks validate that you used the intended Playwright method correctly.
One concept at a time
Each training focuses on a single API or pattern.
Immediate feedback
You see not only if it works, but whether it matches the goal.

Available trainings

GetBy locators

Beginner 12 tasks text / role

Learn how Playwright matches text, roles and semantics. Practice partial vs exact, regex matching and visibility rules.

page.getByText(/^Submit$/)
~20 min

Locator API

Beginner 10 tasks locator()

Understand the base locator API and when CSS selectors are appropriate. Learn to avoid brittle chains.

page.locator('tr', { hasText: /Total/ })
~15 min

Filtering & collections

Beginner 14 tasks filter / has

Work with collections of elements. Learn how filter() and has change matching behavior.

rows.filter({ has: page.getByText('Active') })
~25 min

Advanced basics

Beginner+ 8 tasks edge cases

Transitional set before challenges. Ambiguous text, nested DOM, and trade-offs between readability and robustness.

locator('li').filter({ hasText: /Enabled/ }).first()
~20 min
